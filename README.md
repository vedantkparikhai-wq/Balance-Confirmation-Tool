# AuditCheck Pro Deployment Guide

1. Unzip this folder.
2. Run `npm install`.
3. Copy your latest `App.tsx`, `types.ts`, `utils.ts`, and `services/` folder from the editor into the `src/` folder.
4. Run `npm run dev` to test locally.
5. Deploy to Vercel/Netlify.
6. Add your production domain to Google Cloud Console > APIs & Services > Credentials > OAuth 2.0 Client IDs > Authorized JavaScript origins.
